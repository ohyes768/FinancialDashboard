# 产品需求文档 (PRD)

## 项目概述

**项目名称**：Financial Dashboard（金融数据可视化仪表板）

**版本**：v1.0.0

**最后更新**：2025-11-26

**项目描述**：
本项目是一个金融数据可视化仪表板，用于实时展示和分析关键的金融指标，包括汇率、美国国债收益率、国债规模、GDP、TGA账户余额和HIBOR等数据。通过前后端分离的架构设计，为用户提供直观、交互式的数据可视化体验。

---

## 功能需求

### 1. 核心功能

#### 1.1 汇率数据可视化
- **功能描述**：展示多种货币汇率的相对变化走势
- **数据来源**：CSV 文件（`data/exchange_rates.csv`）
- **展示方式**：折线图，显示相对变化百分比
- **交互功能**：
  - 支持多时间范围切换（1个月、3个月、6个月、1年、3年、全部）
  - 鼠标悬停显示具体数值
  - 图例开关控制显示/隐藏

#### 1.2 美债收益率走势
- **功能描述**：展示不同期限美国国债的收益率变化
- **数据来源**：CSV 文件（`data/treasury_yields.csv`）
- **展示方式**：折线图，显示实际收益率百分比
- **交互功能**：与汇率图表相同的时间范围控制

#### 1.3 美债规模与GDP分析
- **功能描述**：展示美国国债规模、GDP规模及债务占比
- **数据来源**：CSV 文件（`data/treasury_debt_gdp.csv`）
- **展示方式**：
  - 左Y轴：美债规模和GDP规模（万亿美元）
  - 右Y轴：债务占GDP比重（%）
- **交互功能**：双Y轴可视化，时间范围切换

#### 1.4 TGA账户与HIBOR分析
- **功能描述**：展示美国财政部一般账户（TGA）余额与香港银行同业拆息（HIBOR）的关系
- **数据来源**：CSV 文件（`data/tga_hibor_data.csv`）
- **展示方式**：
  - 左Y轴：TGA账户余额（万亿美元）
  - 右Y轴：HIBOR（%）
- **交互功能**：双Y轴可视化，时间范围切换

### 2. 交互功能

#### 2.1 时间范围切换
- **可选范围**：
  - 近1个月
  - 近3个月
  - 近6个月
  - 近1年
  - 近3年（默认）
  - 全部
- **实现方式**：顶部按钮菜单，点击即时切换

#### 2.2 图表交互
- **缩放**：支持鼠标拖拽缩放
- **平移**：支持拖拽平移查看历史数据
- **悬停提示**：鼠标悬停显示详细数据
- **图例控制**：点击图例显示/隐藏特定数据系列
- **重置视图**：一键恢复默认视图

#### 2.3 重大事件标注
- **功能描述**：在图表右侧显示重大事件列表
- **数据来源**：`event.txt` 文件
- **展示格式**：日期 + 事件描述

## 非功能需求

### 1. 性能要求
- **响应时间**：页面加载时间 < 2秒
- **数据更新**：支持实时数据刷新（通过页面重载）
- **并发支持**：支持多用户同时访问

### 2. 兼容性要求
- **浏览器**：支持现代浏览器（Chrome、Firefox、Safari、Edge）
- **分辨率**：支持 1920x1080 及以上分辨率
- **移动端**：优先桌面端体验，移动端响应式布局可选

### 3. 可用性要求
- **界面语言**：中文
- **操作简便**：无需培训即可上手使用
- **错误处理**：友好的错误提示信息

### 4. 可维护性要求
- **日志记录**：完整的操作日志和数据日志
- **配置管理**：数据源路径可配置
- **代码规范**：遵循 Python 和 JavaScript 最佳实践

## 数据需求

### 1. 数据源
| 数据类型 | 文件名 | 更新频率 | 数据格式 |
|---------|--------|---------|---------|
| 汇率数据 | exchange_rates.csv | 每日 | CSV |
| 美债收益率 | treasury_yields.csv | 每日 | CSV |
| 美债和GDP | treasury_debt_gdp.csv | 每日 | CSV |
| TGA和HIBOR | tga_hibor_data.csv | 每日 | CSV |
| 重大事件 | event.txt | 手动更新 | 文本 |

### 2. 数据质量要求
- **完整性**：数据时间序列连续，缺失值自动插值处理
- **准确性**：数据来源于权威机构（FRED、Yahoo Finance 等）
- **时效性**：每日更新数据
