## 1. exchange_rates.csv（汇率数据）

### 文件说明
存储主要货币对美元的汇率数据。

### 数据结构
| 字段名 | 数据类型 | 说明 | 单位 | 示例值 |
|-------|---------|------|------|-------|
| Date | Datetime | 日期（索引列） | - | 2023-01-01 |
| USD/CNY | Float | 美元兑人民币汇率 | - | 7.1234 |
| USD/EUR | Float | 美元兑欧元汇率 | - | 0.9234 |
| USD/JPY | Float | 美元兑日元汇率 | - | 149.56 |
| USD/GBP | Float | 美元兑英镑汇率 | - | 0.8123 |

### 示例数据
```csv
Date,USD/CNY,USD/EUR,USD/JPY,USD/GBP
2023-01-01,7.1234,0.9234,149.56,0.8123
2023-01-02,7.1345,0.9212,149.23,0.8145
2023-01-03,7.1456,0.9198,148.89,0.8167
```

### 数据来源
- **来源**：Yahoo Finance
- **获取方式**：yfinance Python 库
- **对应脚本**：`exchangeRate_data.py`

### 数据处理
1. 读取原始数据
2. 设置 Date 为索引列
3. 转换为 Datetime 类型
4. 处理缺失值（插值填充）
5. 重新索引到完整的工作日序列

## 2. treasury_yields.csv（美国国债收益率数据）

### 文件说明
存储不同期限美国国债的收益率数据。

### 数据结构
| 字段名 | 数据类型 | 说明 | 单位 | 示例值 |
|-------|---------|------|------|-------|
| Date | Datetime | 日期（索引列） | - | 2023-01-01 |
| 2年期 | Float | 2年期美国国债收益率 | % | 4.56 |
| 5年期 | Float | 5年期美国国债收益率 | % | 4.12 |
| 10年期 | Float | 10年期美国国债收益率 | % | 3.98 |
| 30年期 | Float | 30年期美国国债收益率 | % | 4.05 |

### 示例数据
```csv
Date,2年期,5年期,10年期,30年期
2023-01-01,4.56,4.12,3.98,4.05
2023-01-02,4.58,4.15,4.01,4.08
2023-01-03,4.60,4.18,4.04,4.10
```

### 数据来源
- **来源**：FRED（Federal Reserve Economic Data）
- **获取方式**：fredapi Python 库
- **对应脚本**：`usTreasuryBond_data.py`

### FRED 数据代码
- DGS2 - 2年期美国国债收益率
- DGS5 - 5年期美国国债收益率
- DGS10 - 10年期美国国债收益率
- DGS30 - 30年期美国国债收益率

### 数据处理
1. 从 FRED 获取各期限收益率数据
2. 合并不同期限的数据
3. 设置 Date 为索引列
4. 处理缺失值（插值填充）
5. 重新索引到完整的工作日序列

## 3. treasury_debt_gdp.csv（美国国债和GDP数据）

### 文件说明
存储美国国债规模、GDP规模及债务占GDP比重的数据。

### 数据结构
| 字段名 | 数据类型 | 说明 | 单位 | 示例值 |
|-------|---------|------|------|-------|
| Date | Datetime | 日期（索引列） | - | 2023-01-01 |
| tot_pub_debt_out_amt | Float | 美国国债总规模 | 美元 | 3.5e13 |
| GDP | Float | 美国GDP总量 | 美元 | 2.75e13 |
| 债务占比 | Float | 债务占GDP比重 | % | 127.5 |

### 示例数据
```csv
Date,tot_pub_debt_out_amt,GDP,债务占比
2023-01-01,35000000000000,27500000000000,127.27
2023-01-02,35050000000000,27500000000000,127.45
2023-01-03,35100000000000,27500000000000,127.64
```

### 数据来源
- **来源**：FRED（Federal Reserve Economic Data）
- **获取方式**：fredapi Python 库
- **对应脚本**：`usTreasuryDebt_data.py`

### FRED 数据代码
- FYFSGDA188S - 美国国债总规模
- GDP - 美国GDP总量

### 计算字段
- **债务占比**：(国债总规模 / GDP) × 100

### 数据处理
1. 从 FRED 获取国债和GDP数据
2. 计算债务占GDP比重
3. 合并数据到单个DataFrame
4. 设置 Date 为索引列
5. 处理缺失值（插值填充）
6. 重新索引到完整的工作日序列

---

## 4. tga_hibor_data.csv（TGA账户余额和HIBOR数据）

### 文件说明
存储美国财政部一般账户（TGA）余额和香港银行同业拆息（HIBOR）的数据。

### 数据结构
| 字段名 | 数据类型 | 说明 | 单位 | 示例值 |
|-------|---------|------|------|-------|
| Date | Datetime | 日期（索引列） | - | 2023-01-01 |
| TGA | Float | TGA账户余额 | 美元 | 8.5e11 |
| HIBOR | Float | 香港银行同业拆息 | % | 5.23 |

### 示例数据
```csv
Date,TGA,HIBOR
2023-01-01,850000000000,5.23
2023-01-02,855000000000,5.25
2023-01-03,860000000000,5.27
```

### 数据来源
- **TGA来源**：FRED（Federal Reserve Economic Data）
- **HIBOR来源**：Yahoo Finance 或香港金融管理局
- **获取方式**：fredapi / yfinance Python 库
- **对应脚本**：`tgaHibor_data.py`

### FRED 数据代码
- RCPPCNTT01JQM189S - TGA账户余额（需转换）
- 或使用 WTREGEN - Treasury General Account

### 数据处理
1. 从 FRED 获取 TGA 数据
2. 从数据源获取 HIBOR 数据
3. 合并数据到单个DataFrame
4. 设置 Date 为索引列
5. 处理缺失值（插值填充）
6. 重新索引到完整的工作日序列

---

## 5. event.txt（重大事件文件）

### 文件说明
存储金融市场的重大事件，用于在图表上标注。

### 数据结构
每行一个事件，格式为：`日期 事件描述`

### 示例数据
```text
2023-03-10 硅谷银行倒闭
2023-03-12 瑞信被瑞银收购
2023-06-14 美联储暂停加息
2023-09-20 美联储维持利率不变
```

### 使用方式
1. 由 `fastapi_dashboard_backend.py` 读取
2. 解析每行数据，提取日期和描述
3. 在图表上添加注释显示

## 数据处理规则

### 1. 时间索引处理
```python
# 生成完整的工作日时间序列
full_index = pd.date_range(
    start=data.index.min(),
    end=data.index.max(),
    freq='B'  # B = Business days (工作日)
)
```

### 2. 缺失值处理
```python
# 重新索引
data = data.reindex(full_index)

# 时间插值
data = data.interpolate(method='time')

# 前向填充和后向填充
data = data.ffill().bfill()
```

### 3. 数据合并
```python
# 合并所有数据源
data = pd.concat([
    exchange_data,
    treasury_data,
    debt_gdp_data,
    tga_hibor_data
], axis=1)
```

### 4. 相对变化计算（仅汇率）
```python
# 计算相对变化百分比
base_value = period_data[column].iloc[0]
relative_change = (period_data[column] - base_value) / base_value * 100
```
